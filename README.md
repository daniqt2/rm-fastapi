# BACKEND

## Installation

- Install dependencies using `pip`:

  ```bash
  pip install -r requirements.txt

  ```

## Running the Project

Navigate to the `app` directory and run the following command to start the FastAPI application using `uvicorn`:

```bash
cd app
uvicorn main:app --reload
```
